# Jerry AI: Advanced Desktop Assistant

**Jerry** is a voice-activated, intelligent desktop assistant designed for **Windows 11 (Aurora System)**. Built with a focus on productivity, it integrates the **Gemini 2.0 Flash** model for natural language processing and local system automation.

## 🚀 Key Features
* **Voice Commands:** Hands-free control using Google Speech Recognition.
* **AI Brain:** Powered by Google Gemini for intelligent responses and task handling.
* **System Automation:** Opens specific folders, local files, and manages background processes.
* **Web Integration:** Quick shortcuts for LinkedIn, Flipkart, and specialized UPSC/NPTEL searches.
* **Media Control:** Voice-activated YouTube playback and global media "Kill Switch."

## 🛠️ Tech Stack
* **Language:** Python 3.12
* **AI Model:** Google Gemini (Generative AI)
* **Libraries:** `google-genai`, `SpeechRecognition`, `pyttsx3`, `pywhatkit`, `pyautogui`.

## ⚙️ Setup
1. Clone the repository.
2. Create a virtual environment: `python -m venv venv`.
3. Install dependencies: `pip install -r requirements.txt`.
4. Add your **Gemini API Key** to a `.env` file.
5. Run `jerry.py` or `jerry.pyw` for background mode.
